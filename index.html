1. Estructura de un archivo HTML

Todo documento HTML tiene el mismo esqueleto. El navegador lo lee de arriba a abajo. <head> contiene informacion de la pagina (no visible para el usuario). <body> contiene todo lo que el usuario VE. 

<!-- Indica que este archivo es HTML5 -->
<!DOCTYPE html>

<html lang="es">

  <head>
    <meta charset="UTF-8"/>
    <title>Mi pagina</title>
    <link rel="stylesheet" href="estilos.css"/>
  </head>

  <body>
    <h1>Hola mundo</h1>
    <p>Mi primer parrafo.</p>
  </body>

</html>
Importante: Todas las etiquetas abiertas deben cerrarse. Ejemplo: <p> se cierra con </p>. Las etiquetas vacias como <img>, <br>, <meta> no tienen cierre.

2. CSS: Externo, Interno e Inline

CSS es el lenguaje que da color, tamanio y posicion a los elementos HTML. Hay 3 formas de aniadirlo. Prioridad: Inline > Interno > Externo.

1 — CSS EXTERNO (el mas recomendado)

<!-- En el head de tu HTML -->
<link rel="stylesheet" href="estilos.css"/>

/* En el archivo estilos.css */
h1 {
  color: red;
  font-size: 2rem;
}
Ventaja: Un solo archivo CSS puede aplicarse a TODAS las paginas del proyecto a la vez. Si cambias el color en el CSS, cambia en todas las paginas.

2 — CSS INTERNO

<head>
  <style>
    p {
      color: navy;
      font-size: 16px;
    }
  </style>
</head>
Desventaja: Solo afecta a ese archivo HTML concreto. Si tienes varias paginas hay que copiar y pegar los estilos en cada una.

3 — CSS INLINE (en linea)

<p style="color: green; font-weight: bold;">
  Este parrafo es verde y negrita.
</p>
Evitar siempre que sea posible. Mezcla contenido y diseno. Si quieres cambiar el color de 100 parrafos, tendrias que editar 100 etiquetas una a una.

3. Titulos H1 a H6

Los headings son etiquetas de titulo. H1 es el mas grande e importante. H6 es el mas pequenio. Solo debe haber UN h1 por pagina. 

<h1>Titulo principal</h1>
<h2>Titulo de seccion</h2>
<h3>Titulo de subseccion</h3>
<h4>Titulo h4</h4>
<h5>Titulo h5</h5>
<h6>Titulo h6</h6>

Resultado en el navegador

H1 — Titulo principal

H2 — Titulo de seccion

H3 — Titulo de subseccion

H4 — Titulo pequenio

H5 — Mas pequenio
H6 — El mas pequenio

4. Parrafos y formato de texto

<p> es el parrafo basico. Hay etiquetas para dar formato al texto sin usar CSS. 

<p>Esto es un parrafo normal.</p>

<strong>Negrita - texto importante</strong>
<em>Cursiva - texto con enfasis</em>
<u>Texto subrayado</u>
<s>Texto tachado</s>
<mark>Texto resaltado</mark>
<small>Texto pequenio</small>

<br/>   <!-- Salto de linea -->
<hr/>   <!-- Linea horizontal separadora -->

Resultado en el navegador

Esto es un parrafo normal.

Negrita  |  Cursiva  |  Subrayado  |  Tachado  |  Resaltado  |  Pequenio

La linea de arriba es un <hr>

5. Enlaces — <a>

La etiqueta <a> (anchor = ancla) crea un enlace clicable. href indica a donde va. target="_blank" lo abre en nueva pestana. 

<!-- Enlace externo -->
<a href="https://www.google.com">Ir a Google</a>

<!-- Enlace en nueva pestana -->
<a href="https://www.google.com" target="_blank">Abrir en nueva pestana</a>

<!-- Enlace a pagina del mismo proyecto -->
<a href="contacto.html">Ir a contacto</a>

<!-- Enlace a seccion de la misma pagina -->
<a href="#tablas">Ir a la seccion Tablas</a>

<!-- Enlace para enviar email -->
<a href="mailto:alumno@ejemplo.com">Enviar email</a>

Resultado en el navegador

Ir a Google Ir a Tablas Enviar email

6. Botones — <button>

<button> crea un boton clicable. Con CSS se puede hacer de cualquier forma y color. 

<!-- Boton simple -->
<button>Haz clic aqui</button>

<!-- Boton con accion JavaScript -->
<button onclick="alert('Hola!')">Mostrar alerta</button>

<!-- Boton desactivado -->
<button disabled>No se puede clicar</button>

<!-- Boton con estilo CSS inline -->
<button style="background: navy; color: white; padding: 10px 20px; border: none; border-radius: 6px;">
  Boton azul
</button>

<!-- Tipos de boton en formularios -->
<button type="submit">Enviar formulario</button>

Resultado en el navegador (clica los botones)

7. Imagenes — <img>

La etiqueta <img> muestra una imagen. No tiene etiqueta de cierre. src indica la ruta. alt es el texto alternativo (accesibilidad + SEO). 

<!-- Imagen local -->
<img src="fotos/perro.jpg" alt="Un perro simpatico"/>

<!-- Imagen de internet -->
<img src="https://picsum.photos/200" alt="Foto aleatoria"/>

<!-- Imagen con tamanio fijo -->
<img src="foto.jpg" alt="Mi foto"
     width="300" height="200"/>

<!-- Imagen con CSS inline -->
<img src="avatar.jpg" alt="Avatar"
     style="border-radius: 50%; width: 80px;"/>

Resultado en el navegador

Normal
border-radius 50%
Con borde CSS

8. Listas — <ul>, <ol>, <li>

Hay dos tipos: desordenada (puntos) y ordenada (numeros). Cada elemento es un <li>. 

<!-- Lista desordenada = con puntos -->
<ul>
  <li>Manzana</li>
  <li>Naranja</li>
  <li>Platano</li>
</ul>

<!-- Lista ordenada = con numeros -->
<ol>
  <li>Encender el ordenador</li>
  <li>Abrir el navegador</li>
  <li>Ir a Google</li>
</ol>

<!-- Lista anidada (lista dentro de lista) -->
<ul>
  <li>Frutas
    <ul>
      <li>Manzana</li>
      <li>Pera</li>
    </ul>
  </li>
</ul>

Resultado en el navegador

Lista desordenada (ul):
  • Manzana
  • Naranja
  • Platano
Lista ordenada (ol):
  1. Encender el ordenador
  2. Abrir el navegador
  3. Ir a Google

9. Tablas — <table>

Las tablas muestran datos en filas y columnas. tr = fila | th = celda cabecera | td = celda de datos. 

<table>

  <thead>
    <tr>
      <th>Nombre</th>
      <th>Edad</th>
      <th>Ciudad</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Ana</td>
      <td>22</td>
      <td>Madrid</td>
    </tr>
  </tbody>

</table>

<!-- colspan: une celdas horizontalmente -->
<td colspan="2">Ocupa 2 columnas</td>

<!-- rowspan: une celdas verticalmente -->
<td rowspan="2">Ocupa 2 filas</td>

Resultado en el navegador

NombreEdadCiudadNota
Ana22Madrid9
Carlos20Sevilla6
Pedro21Valencia4
Truco CSS: Aniade border-collapse: collapse a la tabla para que los bordes de las celdas no se dupliquen.

10. Formularios — <form>, <input>

Los formularios recogen datos del usuario. El atributo type del input define que tipo de dato acepta. 

<form action="procesar.php" method="post">

  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" placeholder="Tu nombre"/>

  <input type="email" name="email"/>
  <input type="password" name="pass"/>
  <input type="number" name="edad" min="0" max="120"/>

  <!-- Checkbox: casilla de verificacion -->
  <input type="checkbox" name="acepto"/> Acepto los terminos

  <!-- Radio: solo se puede elegir UNO del mismo name -->
  <input type="radio" name="sexo" value="h"/> Hombre

  <!-- select: lista desplegable -->
  <select name="ciudad">
    <option value="mad">Madrid</option>
    <option value="bcn">Barcelona</option>
  </select>

  <!-- textarea: area de texto grande -->
  <textarea name="mensaje" rows="4"></textarea>

  <button type="submit">Enviar</button>

</form>

Resultado en el navegador

11. Clases e IDs

Sirven para identificar elementos y aplicarles CSS. class se puede repetir. id debe ser unico. 

<!-- HTML: asignar clase e id -->
<p class="destacado">Parrafo con clase</p>
<p class="destacado">Otro parrafo con la misma clase</p>
<div id="cabecera">Solo uno con este id</div>

/* CSS: como seleccionarlos */

.destacado {
  color: red;
  font-weight: bold;
}

#cabecera {
  background: navy;
  color: white;
}

/* Multiples clases en un elemento */
<p class="destacado grande azul">Tiene tres clases</p>

Resultado en el navegador

Parrafo con clase .destacado

Otro parrafo con la misma clase .destacado

Div con id="cabecera"

12. Div y Span — contenedores genericos

Son etiquetas sin significado propio que sirven para agrupar y dar estilo. div es bloque. span es en linea. 

<!-- div: contenedor de bloque -->
<div style="background: lightblue; padding: 10px;">
  <h3>Soy un div</h3>
  <p>Puedo contener cualquier cosa.</p>
</div>

<!-- span: contenedor en linea -->
<p>
  El precio es
  <span style="color: red; font-weight: bold;">99 euros</span>
  (oferta limitada).
</p>

/* Etiquetas semanticas: como divs pero con significado */
<header> ... </header>
<nav>    ... </nav>
<main>   ... </main>
<section>... </section>
<article>... </article>
<footer> ... </footer>

Resultado en el navegador

Soy un div

Puedo contener cualquier cosa dentro.

El precio es 99 euros (oferta limitada).

Resumen: <div> para cajas y secciones de layout. <span> para estilizar una palabra dentro de un texto. Usa etiquetas semanticas cuando puedas: <header>, <main>, <footer>, <section>, <article>, <nav>.
