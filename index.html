1. Estructura de un archivo HTML

Todo documento HTML tiene el mismo esqueleto. <head> contiene información de la página (no visible). <body> contiene todo lo que el usuario VE.

<!-- Indica que este archivo es HTML5 -->
<!DOCTYPE html>

<html lang="es">

  <head>
    <meta charset="UTF-8"/>
    <title>Mi página</title>
    <link rel="stylesheet" href="estilos.css"/>
  </head>

  <body>
    <h1>¡Hola mundo!</h1>
    <p>Mi primer párrafo.</p>
  </body>

</html>
Importante: Todas las etiquetas abiertas deben cerrarse. Las etiquetas vacías como <img>, <br>, <meta> no tienen cierre.

2. CSS: Externo, Interno e Inline

CSS da color, tamaño y posición a los elementos HTML. Hay 3 formas de añadirlo. Prioridad: Inline > Interno > Externo.

1 — CSS EXTERNO (el más recomendado)

<link rel="stylesheet" href="estilos.css"/>

/* En el archivo estilos.css */
h1 {
  color: red;
  font-size: 2rem;
}
Ventaja: Un solo archivo CSS se aplica a TODAS las páginas del proyecto a la vez.

2 — CSS INTERNO

<style>
  p {
    color: navy;
    font-size: 16px;
  }
</style>
Desventaja: Solo afecta a ese archivo HTML. Si tienes varias páginas hay que repetirlo en cada una.

3 — CSS INLINE (en línea)

<p style="color: green; font-weight: bold;">
  Párrafo verde y negrita.
</p>
Evitar siempre que sea posible. Mezcla contenido y diseño. Para cambiar el estilo de 100 elementos tendrías que editar 100 etiquetas.

3. Títulos H1 a H6

H1 es el más grande e importante. H6 es el más pequeño. Solo debe haber UN h1 por página.

<h1>Título principal</h1>
<h2>Título de sección</h2>
<h3>Título de subsección</h3>
<h4>Título h4</h4>
<h5>Título h5</h5>
<h6>Título h6</h6>

Resultado en el navegador

H1 — Título principal

H2 — Título de sección

H3 — Subsección

H4 — Pequeño

H5 — Más pequeño
H6 — El más pequeño

4. Párrafos y formato de texto

Etiquetas para dar formato al texto sin usar CSS.

<p>Esto es un párrafo normal.</p>

<strong>Negrita — texto importante</strong>
<em>Cursiva — énfasis</em>
<u>Subrayado</u>
<s>Tachado</s>
<mark>Resaltado</mark>
<small>Texto pequeño</small>
<br/>
<hr/>

Resultado en el navegador

Párrafo normal.

Negrita  |  Cursiva  |  Subrayado  |  Tachado  |  Resaltado  |  Pequeño

5. Enlaces — <a>

La etiqueta <a> (anchor) crea un enlace clicable. href indica a dónde va. target="_blank" abre en nueva pestaña.

<a href="https://www.google.com">Ir a Google</a>

<a href="https://www.google.com" target="_blank">Nueva pestaña</a>

<a href="contacto.html">Ir a contacto</a>

<a href="#tablas">Ir a Tablas</a>

<a href="mailto:alumno@ejemplo.com">Enviar email</a>

Resultado en el navegador

Ir a Google Ir a Tablas Enviar email

6. Botones — <button>

Crea un botón clicable. Con CSS se puede personalizar completamente.

<button>Haz clic aquí</button>

<button onclick="alert('¡Hola!')">Mostrar alerta</button>

<button disabled>No se puede clicar</button>

<button type="submit">Enviar formulario</button>

Resultado en el navegador (clica los botones)

7. Imágenes — <img>

Muestra una imagen. No tiene etiqueta de cierre. src = ruta. alt = texto alternativo (obligatorio).

<img src="fotos/perro.jpg" alt="Un perro simpático"/>

<img src="foto.jpg" alt="Mi foto"
     width="300" height="200"/>

<img src="avatar.jpg" alt="Avatar"
     style="border-radius: 50%; width: 80px;"/>

Resultado en el navegador

Normal
border-radius 50%
Con borde CSS

8. Listas — <ul>, <ol>, <li>

Desordenada (puntos) y ordenada (números). Cada elemento es un <li>.

<ul>
  <li>Manzana</li>
  <li>Naranja</li>
</ul>

<ol>
  <li>Encender el ordenador</li>
  <li>Abrir el navegador</li>
</ol>

<!-- Lista anidada (lista dentro de lista) -->
<ul>
  <li>Frutas
    <ul>
      <li>Manzana</li>
      <li>Pera</li>
    </ul>
  </li>
</ul>

Resultado en el navegador

Lista desordenada (ul):
  • Manzana
  • Naranja
  • Plátano
Lista ordenada (ol):
  1. Encender el ordenador
  2. Abrir el navegador
  3. Ir a Google

9. Tablas — <table>

Muestran datos en filas y columnas. tr = fila | th = cabecera | td = celda de datos.

<table>
  <thead>
    <tr>
      <th>Nombre</th>
      <th>Edad</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Ana</td>
      <td>22</td>
    </tr>
  </tbody>
</table>

<td colspan="2">Ocupa 2 columnas</td>
<td rowspan="2">Ocupa 2 filas</td>

Resultado en el navegador

NombreEdadCiudadNota
Ana22Madrid9
Carlos20Sevilla6
Pedro21Valencia4
Truco CSS: Añade border-collapse: collapse a la tabla para que los bordes no se dupliquen.

10. Formularios — <form>, <input>

Recogen datos del usuario. El atributo type del input define qué tipo de dato acepta.

<form action="procesar.php" method="post">

  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" placeholder="Tu nombre"/>

  <input type="email" name="email"/>
  <input type="password" name="pass"/>

  <input type="checkbox" name="acepto"/> Acepto los términos

  <input type="radio" name="sexo" value="h"/> Hombre

  <select name="ciudad">
    <option value="mad">Madrid</option>
  </select>

  <textarea name="mensaje" rows="4"></textarea>

  <button type="submit">Enviar</button>

</form>

Resultado en el navegador

11. Clases e IDs

Para identificar elementos y aplicarles CSS. class se puede repetir. id debe ser único.

<p class="destacado">Párrafo con clase</p>
<p class="destacado">Otro párrafo con la misma clase</p>
<div id="cabecera">Solo uno con este id</div>

.destacado { color: red; font-weight: bold; }
#cabecera { background: navy; color: white; }

Resultado en el navegador

Párrafo con clase .destacado

Otro párrafo con la misma clase .destacado

Div con id="cabecera"

12. Div y Span — contenedores genéricos

Sin significado propio. div es bloque (ocupa toda la línea). span es en línea (no la corta).

<div style="background: lightblue; padding: 10px;">
  <h3>Soy un div</h3>
  <p>Puedo contener cualquier cosa.</p>
</div>

<p>El precio es
  <span style="color: red; font-weight: bold;">99 euros</span>
  (oferta limitada).
</p>

<header></header>
<nav></nav>
<main></main>
<section></section>
<article></article>
<footer></footer>

Resultado en el navegador

Soy un div

Puedo contener cualquier cosa dentro.

El precio es 99 euros (oferta limitada).

Resumen: <div> para cajas y layouts. <span> para estilizar una palabra dentro de un texto. Usa etiquetas semánticas cuando puedas.
